# Rejin CNC Asset Library

This folder stores selected visual assets for the Rejin CNC demo project.

The assets are intended for export marketing content generation, including blog articles, landing pages, product pages, social posts, email copy, and future AI-assisted workflows.

## Structure

- `images/brand/`: Company, capability, and general brand imagery
- `images/services/`: Service capability imagery
- `images/products/`: Product and application imagery
- `images/cases/`: Case study and customer proof imagery
- `images/materials-process/`: Material, machining process, and finishing imagery
- `image-library.md`: Product-manager-readable index of selected assets

## Maintenance Rule

Keep asset titles, tags, usage notes, and source information in English. The system interface handles Chinese and English UI labels separately.

Images are selected from public Rejin CNC website pages for demo use. Confirm usage rights before publishing these assets externally.
