# Rejin CNC AI Site Test

This directory is an isolated AI website generation test artifact.

## What is included

- `index.html`: Brand homepage
- `services.html`: Manufacturing service page
- `products.html`: Product and application page
- `cases.html`: Case and quality proof page
- `contact.html`: Static RFQ contact page
- `styles.css`: Standalone design system and responsive layout rules
- `assets/images/`: Copied image assets used by the prototype
- `site-structure.json`: Page and component map for AI site testing

Icons are inlined directly in the HTML files so the pages can be opened from the filesystem without an icon sprite request.

## Isolation rule

No existing app entrypoint, navigation, component, build config, or demo data file is required to run this prototype.

To remove the test output, delete only:

```text
ai-site-tests/rejin-cnc-brand-site/
```

## Local preview

Open `index.html` directly in a browser. No dev server, package install, backend, CMS, or WordPress connection is required.

## Source basis

The English content is generated from the local Rejin CNC demo knowledge base: brand profile, audience personas, services, products, solutions, cases, FAQs, and copied image assets.

Images are used for local testing only. Confirm usage rights before any public deployment.
