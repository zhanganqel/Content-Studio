# Docs

这里存放 Content Studio 的项目文档和研发约束。根目录 `README.md` 是项目入口；本目录负责更细的规范和说明。

## 文档索引

| 文档 | 说明 |
| --- | --- |
| `../DESIGN.md` | 项目设计规范，包含 token、组件、icon、Figma/开发一致性规则。 |
| `component-guidelines.md` | 现有共享组件使用规则。 |
| `../AGENTS.md` | AI Agent 协作规则。 |
| `../demo-data/rejin-cnc/README.md` | 演示数据库说明。 |
| `../scripts/README.md` | 支持脚本说明。 |

## 维护规则

- 文档要明确说明适用范围，避免与根 README 重复。
- 设计和组件规范先更新 `DESIGN.md`，再更新组件级说明。
- 后续新增后端、部署、权限等文档时，优先在本目录建立单独文件，并在本 README 登记。
